# Support Generator

Web-приложение для генерации поддержек (решётчатый каркас / egg-crate) под 3D-модель.
Vite + TypeScript + Three.js.

## Запуск

```bash
cd support-generator
npm install
npm run dev
```

Открыть http://localhost:5173

## Деплой на GitHub Pages

В репозиторий нужно положить **содержимое папки `support-generator/`** (она и есть корень
проекта). Дальше:

1. Создать репозиторий на GitHub и запушить туда файлы (ветка `main`).
   Убедитесь, что закоммичены `package.json`, `package-lock.json`, `src/`, `index.html`,
   `vite.config.ts`, `tsconfig*.json` и папка `.github/`. `node_modules` и `dist` —
   в `.gitignore`, их грузить не нужно.
2. В репозитории: **Settings → Pages → Build and deployment → Source = GitHub Actions**.
3. Готово: на каждый push в `main` workflow `.github/workflows/deploy.yml` сам собирает
   проект и публикует. Ссылка появится в **Settings → Pages** и в логе Action
   (вида `https://<логин>.github.io/<репозиторий>/`).

Пути к ассетам относительные (`base: "./"`), поэтому всё работает в подпапке Pages
без дополнительной настройки.

## Что делает

Берёт 3D-модель (сейчас встроенный демо-цилиндр), автоматически определяет тип детали
и строит поддержки:
- **цилиндр/призма** (одинарная кривизна) → поперечные рёбра-сёдла + продольные хребты;
- **двоякоизогнутая** деталь → полная решётка egg-crate (рёбра X + Y).

Ось сетки выравнивается по главной оси детали (PCA). Верхняя кромка каждого ребра
повторяет нижний контур детали; в местах пересечения рёбра врезаются встречными пазами,
снизу — базовые язычки, опционально — облегчающие окна.

Вид по умолчанию — изометрический с автоматическим zoom-fit. Поддержку можно выбрать
кликом и перемещать/вращать гизмо; при наведении подсветка контура.

## Параметры и как они трактуются

Семантика взята из исходного окна; где было неоднозначно — выбран разумный вариант.
Скажите, если что-то нужно понимать иначе — поправлю.

| Параметр | Назначение в текущей реализации |
|---|---|
| Rotate Z | Поворот всей сетки рёбер вокруг оси Z (град) |
| Distance X | Шаг между X-рёбрами (мм) |
| Distance Y | Шаг между Y-рёбрами (мм) |
| Thickness | Толщина материала рёбер (мм) |
| Notch Head Height | Высота опорных «головок» на верхней кромке + зазор-релиф |
| Notch Head Width | Ширина опорной «головки» |
| Notch R1 / R2 | Радиусы скругления насечки (заложены в UI; скругление контура — TODO) |
| Tabs Width / Depth | Размер базовых язычков снизу |
| Slot Height | Глубина врезного паза |
| Slot Width | Ширина паза (≈ толщине материала) |
| Slot R1 / R2 | Радиусы скругления паза (в UI; скругление — TODO) |
| Windows Offset | Отступ облегчающих окон от кромок |
| Generate Windows | Включить облегчающие окна |

## Экспорт

Кнопка **Export STL** — бинарный STL собранного каркаса (в мировых координатах).
